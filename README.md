# 🌦️ Weather API Project

A simple and responsive **Weather Application** built using **HTML5, CSS3, and JavaScript**. This project uses the **WeatherAPI** to fetch and display real-time weather information for any city entered by the user.

## 🌤️ Features

- 🔍 Search weather by city name
- 🌡️ Display current temperature
- ☁️ Show weather condition
- 💧 Display humidity
- 🌬️ Show wind speed
- 📍 Display city and country
- ❌ Error message for invalid city
- 📱 Responsive Design
- ✨ Modern Glassmorphism UI

## 🛠️ Technologies Used

## 🚀 How to Run

1. Clone this repository.

```bash
git clone https://github.com/your-username/weather-api.git
```

2. Open the project folder.

3. Open **index.html** in your browser.

4. Enter a city name.

5. Click the **Search** button.

6. View the live weather information.
